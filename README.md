# Linux Fundamentals – Part 2

This repository contains notes and practical exercises from the **Linux Fundamentals – Part 2** module on **TryHackMe**.

The focus of this module is to strengthen core Linux skills that are essential for cybersecurity, cloud, and infrastructure roles.

## Topics Covered
- Filesystem interaction (`touch`, `mkdir`, `cp`, `mv`, `rm`)
- File type identification with `file`
- Linux permissions and ownership
- Users and groups
- Switching users with `su` and `su -l`
- Linux directory structure (`/etc`, `/var`, `/tmp`, `/root`)

## Key Learnings
- Understanding permissions is fundamental to Linux security
- Access control depends on users, groups, and file ownership
- Flags are discovered through correct access, not guessed
- Knowing where to look is as important as knowing the command
